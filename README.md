# Fruit Classification using CNN

## Overview
This project classifies different fruit images using a Convolutional Neural Network (CNN).

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib

## Features
- Fruit image classification
- CNN model training
- Prediction on new fruit images

## Dataset
Public fruit image dataset used for training and testing.
